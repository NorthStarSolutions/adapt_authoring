Javascript

There are 7 Javascript types:

  1. Number
  2. String
  3. Boolean

Number

Using the 'Modulus %' Number Type, gives the remainder of the sum as the result!

As an example:

  • 12 / 5 = 2.4
  • 12 % 5 = 2 (Rounded Down!)
  • 13 / 5 = 2.6
  • 13 % 5 = 3 (Rounded Up!)

So can carry calculations on the Number Type

ps. The JS Error 'NaN = Not A Number' arises when trying to perform calculations on a mixture of JS types

Theefore ensure use the same JS Types when performing calculations, except when concatenating using '+' symbol

"String"

Use the 'String'JavaScript Type to concatenate words etc

As an example:

  • "This ain't hot Half Mum!"
  • "first_word" added to the "second_word" creates a New word = "first_wordsecond_word"
  • Better to put a space after the first-word
  • "first_word " + "second_word" = "first_word second_word"

  • using the '\' Backslash symbol, means any char immediately after the backslash will be regarded as wysiwyg
  • 'This ain\'t hot Half Mum!'

Boolean True or False

<= (Less than o equal to) >= (Greater than or equal to) === (is Equal to)

For example using the different comparisons:

5 <= 10 = 'True'; 10 >= 5 = 'False'; 20/4 === 5 = 'True'; 20/5 !== 5 = 'True'; 20/4 !== 5 = 'False';
